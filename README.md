# NailsbyNotty

A professional mobile nail technician website designed to showcase services, share information about the business, and provide clients with an easy way to book appointments.

## Overview

This website promotes a house-call nail service, allowing clients to enjoy professional nail care in the comfort of their own homes.

The site includes:

- Home Page
- About Me Page
- Contact & Booking Page
- Service Information
- Nail Design Gallery
- Mobile-Friendly Design

## Features

- Responsive layout for desktop and mobile devices
- Easy navigation
- Professional and modern design
- Service showcase
- Client booking form
- Contact information and social media links
- Portfolio gallery for nail designs
